    Published July 29th 2008

    Welcome to my casual coding universe. I have come to the conclusion I need a place to write down all the tips and tricks I stumble upon while coding away. I never seem to be able to remember the name of that special debug parameter or how to code some very specific validator so now is the time to write this down once and for all. The initial purpose of this blog is to write about technical issues but since I have other interests as well I will probably add posts about travel, photography, cooking, sports and other things too.

    This blog is mainly for my own pleasure but feel free to chime in if you have anything you want to share.
